# tw-utils-api

## About

A tw-utils REST API

## How to build and run ?

1. Get a nodeJS environment (tested with v16.11.1)
2. Install the dependencies `npm i`
3. Create the file .env in the repository source using the .env_example
    - Complete the fields
4. Run the API
    - For production `npm run serve`
    - For dev `npm run dev`
